订单接口

订单接口

客户可以通过订单接口下发入库单、出库单和运单等

简要描述 功能描述
入库单下发 此接口提供给客户系统向鲜易平台发送入库单据
入库单实收反馈 此接口供使用鲜易仓储的客户查询入库实收信息
入库单取消 客户可以通过该接口取消之前发送的入库单指令
出库单接收 此接口提供给客户系统向鲜易平台发送出库单据
出库单发货反馈 此接口供使用鲜易仓储的客户查询出库实发信息
出库单取消 客户可以通过该接口取消之前发送的出库单指令
运单下发 此接口提供给客户系统向鲜易平台发送运输订单
运单取消 客户可以通过该接口取消之前发送的运输订单

结算接口

客户可以通过结算接口获取结算明细

简要描述 功能描述
结算单同步 此接口供客户系统查询运输单的结算信息（按时间范围查询）

仓储接口

客户可以通过仓储接口获取库存量、仓储温度、盘点等信息

简要描述 功能描述
库存查询 此接口供使用鲜易仓储的客户查询库存数据
盘点申请 此接口供使用鲜易仓储的客户下发盘点指令
仓温度查询 此接口供使用鲜易仓储的客户查询货品存储仓库的温度
盘点确认 此接口供使用鲜易仓储的客户确认盘点结果
盘点结果查询 此接口供使用鲜易仓储的客户查询盘点结果

物流接口

物料信息下发到平台。

简要描述 功能描述
物流状态查询 此接口供客户系统查询运输单的全量执行信息
物流状态同步 此接口供客户系统查询运输单的增量执行信息（按时间范围查询）

物料接口

开放给用户交易过程中的快速下单、获取订单接口

简要描述 功能描述
商品下发 通过该接口客户可以将商品明细信息下发到平台

TMS接口

提供伙伴对接TMS系统接口或自由运力平台

简要描述 功能描述
推送运单 平台将运单信息推送到伙伴TMS系统
取消运单 可以使用运单号，通过该接口，取消运单
回传运单状态 伙伴TMS系统回传运单状态变化
回传调度单 伙伴TMS系统执行完调度操作后，回传调度输单
回传应付账单 伙伴TMS系统生成，回传应收账单
回传应收账单 伙伴TMS系统执行完调度操作后，回传调度输单

GPS接口

提供伙伴对接车载设备接口，获取车辆最新位置信息

简要描述 功能描述
获取车辆的最新位置信息 获取当前车辆位置、温度等信息
查询两个地点间的预计时间和里程 获取两个地点间的预计时间和里程
以地图模式获取车辆的最新位置信息 获取车辆的最新位置信息返回地图图片
以地图模式获取车辆的轨迹 获取车辆的最新位置信息返回地图图片

WMS接口

提供伙伴对接WMS系统接口。

简要描述 功能描述
推送入库单 当平台新增入库单时触发该接口，将入库单数据推送到WMS
回传入库单 WMS完成上架操作后触发该接口，将入库单数据回传到平台
回传入库状态 当WMS操作入库作业时，WMS及时将入库单状态推送到平台
取消入库单 当平台取消仓库入库作业的单据时触发该接口
推送出库单 当平台新增出库单时触发该接口，将出库单数据推送到WMS
回传出库单 当WMS完成出库作业后触发该接口，将出库发运数据推送到平台
同步出库状态 当WMS操作出库作业时，WMS及时将出库单状态推送到平台
取消出库单 当协同平台取消仓库出库作业的单据时触发该接口
推送库存全量 WMS定期将指定时间段内的库存变化数据推送给平台
查询库存量 平台向WMS系统查询当前库存量
货品资料 平台新增货品资料信息时触发该接口，协同平台将货品资料推送到WMS
客户资料 平台新增客户资料信息时触发该接口，平台将客户资料推送到WMS
承运商资料 平台新增承运商资料信息时触发该接口，平台将资料推送到WMS
供应商资料 平台新增供应商资料信息时触发该接口，平台将资料推送到WMS

仓控接口

提供伙伴对接仓控系统接口，采集仓库监测点温湿度数据

简要描述 功能描述
推送监测点信息 仓控系统首次推送所有监测点数据到协同平台，后续仓控新增或修改监测点数据时，只需推送新增或修改的监测点数据到协同平台
推送监测数据 仓控系统实时采集监测点温湿度数据，实时将所有监测点温湿度变化数据推送到协同平台
