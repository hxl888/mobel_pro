订单接口>

入库单下发

API描述

此接口提供给客户系统向鲜易平台发送入库单据。

授权类型

  • 需要

应用场景

客户无需导出入库单，客户直接操作其订单系统，通过该接口将入库数据发送到平台。

系统级别输入参数

名称 类型 是否必须 描述 实例值
method String 是 具体API接口名称
requestTime String 是 时间戳，格式为yyyy-mm-ddHH:mm:ss 例如：2013-05-06 13:52:03
appkey String 是 公钥 例如：2013-05-06 13:52:03
secret String 是 密钥
fromSYS String 是 数据来源系统名称
requestContent String 是 传递的报文

表头信息

名称 类型 是否必须 描述 实例值
warehouseCode String Y 鲜易平台的仓库编码
customerCode String Y 鲜易平台为客户分配的客户编码
billType String 采购入库、退货入库、调拨入库
billNo String Y 客户入库单号
carrierCode String 供应商编码
carrierName String 供应商名称
expectedArriveTime String 预计入库时间
correlationNo String 退货入库：原始出库单号
调拨入库：调拨出库单号
notes String 备注

明细信息

名称 类型 是否必须 描述 实例值
itemCode String Y 商品编码
itemName String Y 商品名称
expectedQty Long Y 预期数量
itemUnit String Y 商品单位
notes String 备注

返回结果

名称 类型 是否必须 描述 实例值
billNo String Y 客户入库单号
orderNo String Y 平台入库单号
ynSucceed String Y 是否成功 Y、N
failNotes String Y 失败原因 错误编码
notes String Y 备注

返回示例值

错误码

名称 类型 是否必须 描述 实例值
