物流状态查询

API描述

此接口供客户系统查询运输单的全量执行信息。

授权类型

  • 需要

应用场景

系统级别输入参数

名称 类型 是否必须 描述 实例值
method String 是 具体API接口名称
requestTime String 是 时间戳，格式为yyyy-mm-ddHH:mm:ss 例如：2013-05-06 13:52:03
appkey String 是 公钥 例如：2013-05-06 13:52:03
secret String 是 密钥
fromSYS String 是 数据来源系统名称
requestContent String 是 传递的报文

应用级别输入参数

名称 类型 是否必须 描述 实例值
customerCode String Y 客户编码
billNo String Y 订单编号

返回结果

名称 类型 是否必须 描述 实例值
customerCode String 客户编码
billNo String Y 订单编号
status String Y 操作类型 收货、发运、中转入、中转出、在途、签收
coord String 当前位置 坐标
nowTemp String 当前温度
operator String 操作员
operatorUnit String Y 操作单位 在途：车牌
operatorTime String Y 操作时间
descriptorMessage String 描述信息
发   运：从【操作单位】发车
在   途：搭乘【车牌】运输途中
中转入：【操作单位】收货入库
中转出：【操作单位】发货出库
签   收：货物已签收

返回示例值
